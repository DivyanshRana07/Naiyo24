========================================
API ENDPOINTS USED
========================================

Customers:
  GET    /api/v1/customers         - List all
  POST   /api/v1/customers         - Create
  PUT    /api/v1/customers/{id}    - Update
  DELETE /api/v1/customers/{id}    - Delete

Items:
  GET    /api/v1/items             - List all
  POST   /api/v1/items             - Create
  PUT    /api/v1/items/{id}        - Update
  DELETE /api/v1/items/{id}        - Delete
  PATCH  /api/v1/items/{id}/stock  - Update stock

Invoices:
  GET    /api/v1/invoices/list                - List all
  POST   /api/v1/invoices/create              - Create
  GET    /api/v1/invoices/{id}                - Get details
  PUT    /api/v1/invoices/{id}                - Update
  DELETE /api/v1/invoices/{id}                - Delete
  GET    /api/v1/invoices/{id}/download-pdf   - Download PDF
